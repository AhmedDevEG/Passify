## 🛡️ About Passify
Passify is a modern, offline-first password manager built with Windows App SDK (WinUI 3). Designed with privacy and aesthetics in mind, Passify ensures that your sensitive data never leaves your device. No cloud syncing, no subscriptions, just pure local security.

> **🤖 AI-Developed & Privacy:**
>  Passify was entirely developed by AI, from the first line of code to the last. But don't let "built by AI" make you second-guess your privacy: AI was only involved in writing the app itself, never in handling your data. Everything you store stays encrypted on your device, with no tracking, no collection, and no cloud involved.

## ✨ Current Features (v0.1 Beta)
- **100% Offline & Local:** Your passwords are encrypted and stored locally on your device using SQLite.
- **Beautiful UI:** Modern WinUI 3 interface with Mica backdrop support, fluid animations.
- **Password Generator:** Simple but powerful.
- **Custom Vaults:** Create your own vault file anywhere on your PC and move it around freely.
- **Smart & Custom Icons:** Automatically fetches beautiful website icons powered by [hunter.io](https://hunter.io/api/logo), and fully supports uploading your own custom icons for each password.
- **Import & Export:** Easily back up your vault or restore it to a new installation.
- **Recovery Codes (Out-of-the-Box Feature) 🚀:** A dedicated section to safely store 2FA recovery codes.
  > *You know how websites give you backup codes when you enable 2FA, just in case you lose access or accidentally used a previously used code and forgot to delete it? That's why I built this feature—to keep those backup codes safe for you. 😊😉*

## 🌍 Supported Languages & Translations
Passify aims to be accessible globally. Here is the current translation progress:
- [x] 🇺🇸 **English** (`en-US`)
- [x] 🇪🇬 **Arabic / العربية** (`ar`)*

**Want to translate Passify into your language?** 
Your contributions are highly welcome! To add a new language:
1. Download the English translation template file (`Translation_Template.resw` template in the repository file).
2. Translate the text inside the `<value>` tags to your language.
3. Open a **New Issue** in this repository with the title `Translation: [Your Language]` and attach your translated file.
4. Alternatively, you can send the translated file directly via our Telegram channel.
